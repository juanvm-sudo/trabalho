🌐 Explorando a Web

A internet não é mágica, é infraestrutura. Vamos entender os pilares que sustentam sua navegação diária.

📜 O que é HTTP?

É o idioma oficial da Web. Funciona num sistema de Requisição (você pede) e Resposta (o servidor entrega).

💡 Curiosidade: HTTP significa HyperText Transfer Protocol.

🔐 O que é HTTPS?

É o HTTP com um "colete à prova de balas". Ele usa protocolos TLS/SSL para cifrar os dados.

  • ✅ Protege senhas e cartões
  • ✅ Garante que o site é autêntico

💻 O que é Cliente?

É a ponta final. O seu navegador (Chrome, Safari) atua como o cliente que interpreta códigos (HTML/CSS) para você.

🖥 O que é Servidor?

Computadores potentes que ficam ligados 24/7. Eles "servem" os arquivos que o cliente solicitou.

🎉 Você subiu de nível!

Agora você domina o fluxo básico da comunicação digital.
